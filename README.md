# ServerX

基于Spring 和 Netty 实现的 Web服务框架，由于Netty具有NIO、多路复用等特性，该框架和普通容器（Tomcat）相比，拥有更好的性能。

### 相关组件

* System Define（Default、Annotation）
* Startup（Load Config、Bind address）
* Configuration(Filter、init、web)
* Spring（Factory、MVC）
* Netty（HTTP、TCP）
* XTools（XML、JSON）

### 系统优化
* Linux 优化
* JVM 优化
* Code 优化
